## Règles d'évaluation

- Score sur le comportement produit, PAS sur le style de code ni les noms de variables
- Un écart de nommage (ex: `noResultsRow` au lieu de `noResults`) n'est jamais une erreur
- Une fonctionnalité partiellement implémentée reçoit des points partiels, pas 0
- Si une fonctionnalité est absente, le score du critère est 0
- Ne te base pas sur ta connaissance de ce que le plugin "devrait" faire : base-toi uniquement sur le comportement décrit dans la grille ci-dessous

## Grille de scoring (100 pts)

**1. Activation sans fatal error (15 pts)**
Le fichier principal PHP est syntaxiquement valide, définit `ABSPATH` guard, et le plugin pourrait s'activer sans erreur fatale. Vérifier : `if ( ! defined( 'ABSPATH' ) ) exit;`, constantes définies, classe instanciée.

**2. Filtrage DOM fonctionnel (20 pts)**
Le JS intercepte l'input natif de GF (`#form_list_search` + `input[name="s"]` ou équivalent), filtre les lignes `<tr>` du `tbody#the-list` en les masquant/affichant sans rechargement de page, et utilise un debounce (tout délai entre 100ms et 300ms est valide).
- 20 pts : filtrage + debounce + cache du texte de recherche par ligne
- 14 pts : filtrage + debounce, sans cache
- 8 pts : filtrage fonctionnel mais sans debounce
- 0 pt : AJAX côté serveur, ou jQuery form submit, ou filtre absent

**3. Condition de chargement stricte (10 pts)**
Les assets ne sont chargés que sur la page liste GF. Vérifier côté PHP que la page est `gf_edit_forms` ET qu'on n'est pas sur l'éditeur d'un formulaire spécifique (absence de `$_GET['id'] > 0`).
- 10 pts : double condition (page + absence d'id formulaire)
- 6 pts : condition sur `gf_edit_forms` seulement
- 0 pt : chargement sur toutes les pages admin ou condition absente

**4. Row "no results" (10 pts)**
Une ligne de type "aucun résultat" est injectée dans le DOM et affichée uniquement quand le filtre actif ne correspond à aucune ligne. Elle doit disparaître quand le champ est vidé.
- 10 pts : comportement complet (apparaît / disparaît / texte traduit ou statique)
- 5 pts : présente mais ne disparaît pas correctement
- 0 pt : absente

**5. Raccourcis clavier (10 pts)**
Un raccourci clavier focus l'input de recherche. Ignorer si le focus est déjà dans un champ éditable (input, textarea, select, contenteditable).
- 10 pts : Ctrl/Cmd+F ET touche `/`
- 6 pts : l'un des deux seulement
- 3 pts : raccourci présent mais sans garde sur les champs éditables
- 0 pt : absent

**6. Mise à jour du compteur (10 pts)**
Le compteur natif GF (`.displaying-num`) est mis à jour en temps réel pour refléter le nombre de résultats visibles. Il est restauré à sa valeur originale quand le champ est vidé.
- 10 pts : mise à jour + restauration
- 6 pts : mise à jour sans restauration
- 0 pt : absent

**7. Préchargement des pages paginées (10 pts)**
Si la liste GF est paginée (plusieurs pages), le JS charge en arrière-plan les autres pages via `fetch` pour que le filtre opère sur tous les formulaires, pas seulement la page courante.
- 10 pts : `fetch` + parsing HTML + injection des lignes dans le DOM
- 4 pts : mécanisme de pagination présent mais incomplet ou utilisant AJAX WP
- 0 pt : absent (filtre limité à la page courante seulement)

**8. Diacritiques et casse (5 pts)**
La comparaison est insensible à la casse ET aux diacritiques (`é` trouve `e`, `É` trouve `é`).
- 5 pts : `.toLowerCase()` + `.normalize('NFD')` + suppression des accents
- 2 pts : insensible à la casse seulement
- 0 pt : comparaison brute

**9. Internationalisation (5 pts)**
Les chaînes visibles par l'utilisateur sont traduisibles.
- 5 pts : `__()` / `_n()` côté PHP + mécanisme de passage des traductions au JS (`wp_add_inline_script` ou `wp_localize_script`)
- 2 pts : `__()` côté PHP seulement, JS en dur
- 0 pt : aucune i18n

**10. Qualité du code PHP (5 pts)**
- 5 pts : classe singleton, `defined('ABSPATH')`, constantes plugin (DIR/URL/VERSION), hook `admin_enqueue_scripts` (pas `wp_enqueue_scripts`)
- 3 pts : 3 des 4 éléments présents
- 1 pt : code procédural fonctionnel mais sans les patterns WP standard
- 0 pt : code invalide ou inutilisable
